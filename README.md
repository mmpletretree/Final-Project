# Final-Project

Dungeon Generation with Union-Find
Research Question
How does the choice of Union-Find optimization (path compression, union by rank, both, or neither) affect the total number of operations required to produce a fully connected dungeon of n rooms?
Project Structure
project/
    code/
        experiment.cpp    - C++ implementation of all four Union-Find variants and experiment runner
        plot.py           - Python script to generate plots from CSV data
    data/                 - CSV files with experimental results (generated by experiment)
    plots/                - PNG plots (generated by plot.py)
    reproduce.sh          - Script to reproduce all results
    README.md             - This file
How to Reproduce
bashchmod +x reproduce.sh
./reproduce.sh
This will compile the C++ code, run all experiments, and generate plots.
Requirements

g++ (C++17 or later)
Python 3 with matplotlib
